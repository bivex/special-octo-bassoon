# special-octo-bassoon

Step-by-step guide for building an API from scratch, with OpenAPI plugin configuration.

## Contents

| File | Description |
|------|-------------|
| `stupid_steps.md` | Step-by-step API creation guide |
| `openapi-plugin-manual.md` | OpenAPI plugin documentation |
| `openapi-plugin-manual-base.md` | Base OpenAPI plugin reference |
| `sonaropenapi-rules-1.2.3 for 2025.3.0.jar` | SonarQube OpenAPI rules plugin |

## Getting Started

Follow `stupid_steps.md` for a hands-on walkthrough of creating an API from scratch.

## OpenAPI Tools

The repository includes a SonarQube plugin for OpenAPI rules validation (`sonaropenapi-rules-1.2.3 for 2025.3.0.jar`) to enforce API quality standards.
